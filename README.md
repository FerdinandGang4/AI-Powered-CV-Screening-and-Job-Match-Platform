# AI-Powered CV Screening and Job Match Platform

An academic and practical project for building an AI-powered recruitment platform using ASP.NET for backend services and React for the frontend interface.

## Project Summary

The system helps recruiters:

- upload a job description
- upload multiple CVs
- parse CV content
- extract skills, years of experience, and projects
- match candidates against role requirements
- score and rank candidates
- explain ranking decisions
- highlight missing skills and gaps

## Technology Stack

- Backend: ASP.NET Web API
- Frontend: React.js with Vite
- Architecture direction: modular and scalable
- Future extension: AI/ML-assisted scoring and semantic matching

## Repository Structure

- `Backend_aspdotnet`
  ASP.NET backend project
- `frontend_reactjs/FrontEnd`
  React frontend project
- `documentation`
